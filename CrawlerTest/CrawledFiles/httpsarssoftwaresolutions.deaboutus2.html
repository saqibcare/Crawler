Unsere kreativen Köpfe

Unser Team besteht aus professionellen Programmierern und Designern, die jahrelange Erfahrung mit modernen Technologien haben.
ARS wird aus dem Lateinischen mit Kunst übersetzt, unsere Kunst ist es mit Leidenschaft zu Programmieren.

Unsere Kompetenzen:
Microsoft Technologies (Asp.Net core, Razor Pages, MsSQL)
Cloud Services (Azure Cloud, AWS Cloud, Google Cloud)
FrontEnd React, Angular
Mobile Development mit Flutter
Python (Django, Django RestFramework)
WordPress
Machine Learning (Numpy, TensorFlow, Scipy, Matplotlib, Keras)
